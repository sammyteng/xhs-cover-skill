# 小红书AI生图封面skill (XHS AI Cover Generator)

直接生成或修改小红书封面。本工具为通用 Agent Skill，不绑定特定客户端。支持配置多模态大模型 API Key（如 Gemini 或 OpenAI 兼容接口）进行独立生图，也可以在 Codex 等环境下配合客户端内置的图片生成能力使用。

---

## 🎯 目标

用户调用本 skill 后，预期达到的状态：
1. **生成高质量封面**：成功生成或局部修改小红书 3:4 / 1:1 / 9:16 / 4:3 比例的封面图片，并保存在指定的输出目录中。
2. **人像真实与融合**：保持原始人脸/人像的五官、发型、身份特征完全不变，只改变构图、背景融合或文字叠加。
3. **精美文本排版**：封面上叠加的标题及副标题的字体字形、包边、阴影效果完美契合所选的风格模板，中文无乱码且无多余错乱字符。
4. **统一资产交付**：生成的文件体积经过 EXIF 校正与智能压缩，全部小于 4MB，可流畅上传至小红书。

---

## ✅ 验收清单

本 Skill 在以下条件全部满足前**不得返回**给用户：
- [ ] 目标输出目录下生成了至少 1 张合法的图片文件（`.jpg` 或 `.png`）。
- [ ] 生成图片的画幅比例符合用户指定要求（默认 3:4 竖版）。
- [ ] 生成的图片文件大小小于 4MB。
- [ ] 主标题和副标题文本与输入完全一致，中文字体渲染正确，且没有生成未要求的随机英文或文字。
- [ ] 若为“修改/编辑模式”，原图人物特征、面部五官、构图主体基本得到保留，只进行了用户指明的局部调整，未从零重构。

---

## 📥 输入

- `image`（必填，String）：人物照片路径（支持 JPG/PNG/WebP），或上一版封面的本地绝对路径（修改模式）。
- `style`（必填，String）：风格 ID，可选 22 种内置风格（如 `tech-finance-analytical`、`hand-drawn-border` 等）或用户自定义的 JSON 风格名。
- `title`（必填，String）：封面最瞩目的主标题（最大字）。
- `subtitle`（可选，String，默认空）：辅助副标题（较小字，分行排版）。
- `extra`（可选，String，默认空）：对风格、构图的额外细节或摆放要求。
- `count`（可选，Number，默认 1，最大 5）：单次生成的张数。
- `aspect-ratio`（可选，String，默认 `3:4`）：画布比例（`3:4` / `1:1` / `9:16` / `4:3`）。
- `output-dir`（可选，String，默认 `~/Desktop/XHS封面`）：封面保存的目标目录。

---

## 🔁 工作流

1. **理解目标**：在 stdout 中复述本次生成的目标风格（如 `tech-finance-analytical`）、主标题、副标题和验收清单。
2. **环境及配置读取**：
   - 读取 `~/.config/xhs-cover/config.json`，加载 API Key、Base URL、Model 等 API 凭证。
   - 检查本地是否已安装图片处理库 `sharp`，若未安装则自动执行安装：`npm install sharp`。
3. **图像预处理**：
   - 运行 `generate.mjs` 中的 `normalizeImage` 对人物照进行 EXIF 自动纠偏、旋转和尺寸最大 1920px 缩放。
   - 若处理后的图片大于 4MB，使用 `compressImage` 进一步轻度有损缩水，直至符合验收标准。
4. **构建多模态 Prompt 并调用 API**：
   - 从 `styles/` 中加载目标风格的 JSON 模板。
   - 拼装全局禁止事项（禁止多余文字、禁止修改人脸）与输入文字，调用图片生成 API（Gemini 3 Pro Image / OpenAI-like Image API）。
5. **循环自检与修复**：
   - 自循环重试最多 2 次。如果 API 响应超时、无有效图片数据，或遭遇 5xx 服务端错误，则等待延迟后自动重试。
   - 保存最终图片至目标目录。
6. **展示与交付**：展示生成的图片文件绝对路径及预览。

---

## ⚙️ 副作用

- **写入路径**：`~/.config/xhs-cover/config.json` (配置)，`~/Desktop/XHS封面` (默认输出目录，或用户指定的 `--output-dir`)。
- **调用 API**：OpenAI 兼容的 `v1/chat/completions` 图片生成接口，或 Google AI Studio 原生 `generateContent` 接口。
- **使用凭证**：从 `~/.config/xhs-cover/config.json` 中读取 `apiKey`。
- **破坏性操作**：否。

---

## 🚨 失败回执

- **控制台错误流**：打印具体的错误阶段与错误类型（如 `AUTH_ERROR` 凭证失效、`TIMEOUT` 请求超时、`NO_IMAGE` 格式错误等）。
- **退出码**：
  - `0`：全部生成成功
  - `1`：参数配置或本地模块缺失
  - `2`：API 认证失败 (401/403)
  - `3`：API 连接超时
  - `4`：网络传输错误
  - `5`：未获得可识别图像
